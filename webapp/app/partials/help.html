Help section

In this section we are going to explain what does the data in .par and .tim files represent. A tim file refers to arrival time file. It is an observation file and contains the pulsar arrival time at the observatory.

Example of a NANOGrav 5-year data .tim file (J0030+0451 pulsar)

Here is a row from the J0030+0451 pulsar .tim file. We are going to explain what each column represents:

53358.000077.3.000.000.tsum  420.000  53358.9598798347905  1.820 ao -fe 430G -be ASP -B 430 -bw 4.0 -tobs 2401.070 -pta NANOGrav -proc dfg+12 -chanid asp_420

.tim file column Explanation
53358.000077.3.000.000.tsum 53358' refers to the pulsar file name. '3' refers to the observatory number
420.000 observing frequency in MHz
53358.9598798347905 measured TOA (Time of Arrival at the observatory)
1.820 error in microseconds
ao radiotelescope used for observation ('ao' = Arecibo Observatory)
-fe front-end of the receiver
430G 430 MHz Gregorian
-be back-end of the receiver
